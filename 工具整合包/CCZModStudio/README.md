# CCZModStudio 6.5

新的曹操传加强版 6.5 MOD 集成编辑器原型。目标是把现有工具的核心能力逐步重写进一个统一 Windows 桌面程序，而不是旧工具启动器或脚本集合。

当前开发主线已经收敛为核心可写编辑器：已完成 `角色设定`、`兵种设定`、`地形编辑`、`地图制作` 的可写主流程，并已进入 `战场制作` 与 `剧本制作` P0。战场页具备标题/胜败条件写回、地图预览、出场/坐标候选和战场备注；剧本页具备 Scene/Section/Command 树、命令列表、参数表格、文本线索、注释详情、安全文本写回和旧版完整树下的保守命令增删。后续继续确认字段级写回，再推进 `形象设定深化` 和发布封装。诊断、报告、探针和制作向导保留为高级辅助，不再作为近期新增主线。

## 当前能力

已打通以下集成模块：

- 核心创作首页：启动后优先进入角色设定、兵种设定、地形编辑、地图制作、战场制作、剧本制作、形象设定七个创作入口；诊断、报告、补丁和各类探针集中到“高级工具”。
- 角色设定：独立角色编辑页合并 `6.5-0 人物`、`6.5-0-4 R形象`、`6.5-0-5 S形象`，可编辑名称、头像、台词编号、Army、五维、初始 HP/MP、职业、等级、经验和 R/S 形象编号；职业列使用 `6.5-4 详细兵种` 下拉，附带职业名称、头像说明、筛选、中文字段注释、R/S 资源缺失高亮、保存前备份和保存后复读。右侧角色文本区可联动编辑人物列传、暴击台词和撤退台词，写回 `Imsg.e5`。
- 兵种设定：独立兵种编辑页包含 `详细兵种`、`兵种系/地形`、`兵种特效` 三个子页。详细兵种合并 `6.5-4 详细兵种`、`6.5-4-1 兵种说明`、`6.5-4-2 兵种成长`、`6.5-4-3 兵种穿透`；兵种系/地形合并 `6.5-3 兵种系`、`6.5-3-1 地形发挥`、`6.5-3-2 移动消耗`；兵种特效合并 `6.5-7 兵种特效`、`6.5-7-1 兵种特效说明`、`6.5-7-2 兵种特效分配`，可编辑说明、武将/兵种分配和特效值，并显示只读武将名/兵种名。支持筛选、中文字段说明、GBK 字节容量提示、数值输入校验、保存前备份和保存后复读，写回 `Ekd5.exe`、`Data.e5`、`Imsg.e5`。`6.5-7` 特效名称行存在未拆分尾部字节，当前只读显示，不开放改名。
- 地形编辑：新增可写 `Hexzmap.e5` 地形格编辑器。读取后可选择 `Mxxx` 地图块，使用 0..255 画笔 ID 点击地形格修改；地形格尺寸为变量尺寸，按同编号地图底图 `Map\\Mxxx.jpg` 分辨率 / 48 计算（48x48 像素/格）。画笔会显示素材库 hex 标记解析出的中文地形候选，可选择纯地形色块或叠加同编号地图底图预览。保存前检查 6.5 `Hexzmap.e5` 段长度与底图格数一致性、自动备份，只写当前地形块数据区，不改文件头和尾部未解释字节，保存后复读并生成结构化 `Hexzmap地形格写入` 报告。
- 地图制作：`Map\\Mxxx.jpg` 底图与 `Hexzmap.e5` 变量尺寸地形层已整合到同一创作画布。支持缩放、地形叠加、网格、透明度、鼠标格坐标、中文地形候选、地形画笔、撤销/重做、保存地形层、导出预览 PNG，并可替换当前 JPG/JPEG 底图；底图替换保存前备份，保存后复读校验尺寸和 SHA256。
- 战场制作：独立战场页可读取 SV 关卡，联动地图底图和 Hexzmap 地形层，编辑并保存标题/胜败条件文本；下方按创作者视角提供 `出场/坐标候选` 与 `战场命令定位` 两个表，集中显示人物/部队、坐标、阵营、AI、等级/状态候选和中文注释。新增 `记录战场备注`，可把当前关卡与选中候选保存到项目侧备注中。当前命令结构候选只读，已确认文本按保存前备份、保存后复读写回。
- 剧本制作：新增核心剧本页，按 CczSceneEditor2 习惯显示 Scene/Section/Command 树、命令列表、文本线索、参数/注释详情和文本编辑框；命令列表显示命令号、命令名、偏移、参数预览、参数模板、引用候选和中文注释；右侧参数表格会把 P1/P2/... 拆成参数名、类型、原始值、候选解释、含义和风险边界；支持搜索后定位到左侧 Scene/Section/Command 树，并可复制含参数表和安全边界的命令摘要。旧版完整树模式下可从 `CczString.ini` 选择普通命令，添加到正文末尾、在普通命令前后插入、追加到已有子块或删除未被跳转引用且无子块的普通命令；选中普通数值参数槽后可输入十进制或 `0x` 十六进制并应用，文本参数仍走文本节点编辑。正文根、子事件设定/载体、结束标记、`0x76` 跳转和可变数组等结构/跳转参数仍受保护。选中文本后可在原容量内 GBK 短写回，保存前备份，保存后复读；可为命令或文本创建项目侧剧本备注。
- 当前项目直接保存：已确认 HexTable 数据表和人物 R/S 形象联动允许直接保存当前 MOD 项目，保存前自动备份目标文件，保存后重新读取校验；测试副本仍可作为保护模式使用。
- 本地 MCP 接入：`CCZModStudio.McpServer` 提供 stdio MCP Server，可无头执行项目检测、项目体检、资源索引、资源诊断、创建测试副本、测试副本差异、生成发布副本、表格读写、R/S 剧本文本读写、R/S eex 文件列表/命令结构读取/脚本搜索、R/S eex 命令模板查询、本地知识库搜索、Hexzmap 地形块读取/地形格写入、地图底图替换、非核心资源替换、本地知识库读取，以及 E5 图片资源 `0x110` 索引表读取/预览/单条替换；E5 写入拒绝核心表文件，仍走备份、复读和结构化报告，命令模板与命令结构查询只作只读核对不作为完整剧本结构写回依据。
- 项目识别与文件检查：自动识别 `加强版6.5未加密版`、核心 `.e5`、`Ekd5.exe` 和资源目录。
- 项目体检/发布检查：检查安全模式、核心文件尺寸/SHA256、关键目录、启用表偏移、资源数量，并可导出报告；差异/发布页可一键生成“发布前综合报告”，把核心文件状态、项目体检、测试副本差异、备份历史、结构化写入报告摘要、MOD创作风险摘要和发布前清单汇总成 Markdown。
- 制作向导最近证据：制作向导会扫描工作区报告、导出物、测试副本差异报告和备份写入报告，集中显示发布前综合报告、关卡地图联动报告、SV 命令引用核对清单、预览 PNG、CSV 和结构化写入报告；支持刷新、打开、定位，并可抓取为“报告/发布证据”创作者备注。
- CczRSX 表结构读取：解析 `CczRSX 6.5\ConfigTable\HexTable.xml`。
- 数据表编辑：显示、编辑、校验并保存启用的 6.5 表；支持对当前表数值列绘制分布直方图。
- 保存闭环：已确认数据表/R-S 形象可直接保存当前项目；保存前备份，保存后复读校验；补丁、资源替换、SV 短文本等高风险写入仍按模块规则控制。
- 测试副本差异/发布检查：测试副本可与 `_CCZModStudio_TestCopy.txt` 中记录的原始项目做文件级 SHA256 比较，支持按差异状态和关键字筛选，导出发布前差异报告；选中差异文件时会自动关联同路径备份时间线，显示最近备份、来源动作、回滚建议和最近结构化报告中文摘要，也可一键把“备份历史/回滚”页筛选到相关备份；确认后可一键生成干净发布副本，自动排除测试副本标记、备份、报告和导出目录，并写入发布清单。
- 备份历史/回滚：读取测试副本 `_CCZModStudio_Backups`，按时间线列出表格保存、R/S 保存、剧本文本写回、补丁应用、资源替换/还原产生的备份；能显示目标路径、备份大小、关联报告、来源动作和中文说明；可定位备份文件、打开报告、导出 CSV，并对可还原项执行带预览/再备份/报告的整文件回滚。
- 结构化写入报告：数据表保存、人物 R/S 形象保存、SV 剧本文本写回、资源整文件替换/还原和普罗补丁应用都会在备份目录生成 `*_WriteOperationReport.json`，记录目标文件、备份、前后 SHA256、变化字节、表名/行号/字段名、SV 文本偏移、补丁地址/文件偏移、资源来源、原值/新值和中文说明；备份历史会自动识别这些 JSON 报告并优先使用 JSON 而不是同备份 TXT 报告，选中相关备份时会直接在信息框显示中文“具体改动”摘要，便于从“文件差异”追溯到“具体改了什么”。
- 创作者备注/注释层：新增“创作者备注”页，备注保存在工作区 `CCZModStudio_Notes` 的项目侧 JSON 中，不写入任何游戏文件；支持读取、筛选、抓取当前选择（数据表单元格、SV 命令、SV 文本、资源、EEX、Hexzmap、关卡地图联动、差异/备份等）、编辑保存、删除、导出 CSV，并可点击“定位备注目标”反向跳转到对应表格/SV/资源/Hexzmap/联动/备份页面；用于记录剧情设计、资源用途、修改理由、风险、待办和实机验证结果；JSON 每次覆盖前会自动生成唯一 `.bak.json` 备份，避免连续保存冲突。
- CSV：当前表导出/导入。
- 文本表：通过统一表格机制编辑 `Imsg.e5` 固定长度 GBK 文本。
- 人物 R/S 形象联动：`形象设定` 页面合并人物名、职业、R 形象编号、S 形象编号，可直接写回当前项目 `Ekd5.exe`，保存前备份、保存后复读；资源状态按 `Pmapobj.e5` 与 `Unit_atk/mov/spc.e5` 定位，不再把 `RS\R_XX.eex` / `S_XX.eex` 当作人物图像封包。头像、R、S 预览均按 E5 文件 `0x110` 起始的 12 字节大端索引表取图：R=n 取 `Pmapobj.e5` 图 `2n+1`；S 使用紧凑编号映射，`S=0` 按职业和预览阵营取默认兵种图，`S=1..32` 取三转特殊三张图，`S>=33` 取一转特殊单张图；BMP/JPG/PNG 直接解码，原始帧条按固定尺寸裁代表帧并套用 `tsb` 256 色调色板，禁止裸扫候选图。`导入/替换E5` 已开放单个图片索引条目写入：先从 R/S 映射候选中选择具体 E5 文件和图号，再导入 BMP/JPG/PNG 或从备份 E5 抽取同图号还原；保存前备份目标 E5，保存后复读索引和条目字节，不重排其它条目。
- 普罗补丁导入：解析补丁版本、OD/UE 地址类型、补丁项；OD 地址通过 PE ImageBase/Section 映射到文件偏移；支持预览并应用到测试副本；写入后生成 TXT 报告和结构化 JSON，记录每个补丁项的地址、文件偏移、原字节和新字节。
- 普罗批量搬运配置预览：解析源偏移、目标偏移、长度并生成表格预览；当前不执行跨版本写入。
- 剧本字典预览：读取 `a新剧本编辑器v0.23\CczString.ini`，建立命令 ID -> 名称字典和参数字符串组预览。
- 普罗素材库索引：索引 `普罗-综合工具v0.3\素材库` 图片、hex 标记、说明，并提供图片预览。
- 游戏资源索引：索引根目录、`E5`、`Map`、`RS`、`SV`、`WAV`、`SoundTrk`，并预览地图 JPG；支持分类下拉、关键字筛选、定位选中文件并导出完整资源索引 CSV。测试副本中支持资源整文件替换预览、SHA256/大小/改动字节比较、格式检查、自动备份、TXT+JSON 报告生成和从备份还原；图片资源在确认替换/还原前会以左右并排缩略图显示当前目标与来源文件；WAV/MP3 会显示声道、采样率、位深/码率和估计时长等中文参数。`地图浏览`页支持地图列表、适应窗口、100% 和 10%-300% 缩放。
- 资源格式识别：游戏资源索引会读取文件魔数并标记 `EEX资源包`、`Ls压缩资源`、`JPEG地图/图片`、`SV剧本/关卡`、`WAV音效`、`MP3音轨`、`Windows PE` 等格式线索。
- 资源诊断：基于游戏资源索引生成中文异常报告，覆盖分类概览、重复编号、连续编号缺口、命名不规则、空文件、格式线索异常和地图图片尺寸不可读；同时合并人物 R/S 形象表引用诊断、数据表跨表引用诊断和关卡地图联动诊断，提示“人物 R/S 编号未定位到 Pmapobj.e5/Unit_*.e5”“人物/物品/兵种/商店/专属装备编号需要复查”“多个已命名人物共享同一形象”“R/S 剧本 EEX 缺号或格式异常”“SV/Map/Hexzmap 不成套”等创作风险；支持 Error/Warn/Info 级别筛选、分类筛选、关键字搜索、选中行查看证据与创作建议、导出当前诊断 CSV，并可把选中诊断反向定位到“游戏资源索引”、人物R/S形象、数据表具体行/字段、关卡地图联动、SV 探针、Hexzmap 地形块或地图浏览。
- 资源替换后诊断联动：在测试副本中替换资源或从备份还原后，会自动刷新资源诊断，并在资源信息框追加 Error/Warn/Info 数量和与当前资源可能相关的诊断摘要；若变更对象是地图图片、SV 剧本或 Hexzmap 相关资源，还会刷新关卡地图联动证据并提示导出检查报告、预览PNG和实机复查。
- EEX 资源探针：只读扫描 `RS/*.eex` 与 `Map/*.eex`，识别 `EEX\0` 魔数、版本、疑似条目数、头部字段与 GBK 文本线索；支持分类筛选、关键字搜索、定位选中文件和导出当前 EEX 索引 CSV。选中 EEX 后可生成只读“区段/帧表候选”探针，显示头字段、候选区段、偏移范围、字节统计、文本线索、动作参数/压缩载荷候选和中文说明；同时提供“区段/动作候选树”，按文件头、头字段、文本线索、动作参数/帧表、图像/压缩载荷、透明/稀疏数据等角色分组，点击节点会显示创作者解释、建议操作和安全边界；还能对整文件或选中区段生成原始字节热力图，显示 00/FF 占比、熵、高频字节/16 位词、文本线索和候选解释，并可导出 PNG；“跨文件对比”页会把同编号 R/S 与同分类邻近 EEX 的区段角色、长度、00 占比、小整数比例和文本线索放在一起，帮助判断哪些区段像共同结构、哪些像特殊动作或额外载荷。当前不解压、不还原帧图、不重封包写入。
- SV 剧本/关卡探针：扫描 `SV/*.E5S`，显示标准/扩展关卡分类、16 位字流统计、占用范围、命令候选、标题/文本线索；剧本文件列表支持类型筛选、关键字搜索、仅有文本、定位选中文件和导出当前 SV 索引 CSV；可对选中文件生成命令候选 DataGrid、结构草图 DataGrid、事件树 TreeView、节点详情说明、XML 文本视图和文本线索 DataGrid；结构草图表格会显示“有模板”和“命令参数模板提示”，事件树节点也会标记“有参数模板”；结构草图/XML 页提供关键字、仅有模板、文本/剧情、地图/坐标、高风险/需核对筛选，命中命令时保留对应场景/段落父节点；高风险筛选会给出具体原因，重点收录流程/变量/分支、奖励/资源/人物状态变更、缺模板但影响地图/资源的命令，不再把全部“需实机验证”的普通命令都列入；点击事件树节点时会显示中文注释、参数预览、16 位参数分组解释、常见命令参数模板（如变量测试、对话、地图视点、武将进点/进区域、获得物品、旁白等）、人物/物品/策略/坐标/地图编号候选、同文件文本线索、关卡地图联动和创作提示，并可导出剧本文本 CSV/TXT 或结构 XML。
- SV 命令引用核对：结构草图/XML 页提供 `命令引用` 下拉框、`跳到命令引用` 和 `导出命令引用清单`。可把当前命令后续 16 位词中的疑似人物、物品、策略、文本、地图/坐标整理为可跳转候选，并导出 Markdown 核对清单；清单包含总览、优先核对、命令明细、推荐工作流和安全边界，只写入 `CCZModStudio_Reports`，不修改游戏文件。
- SV 剧本文本安全写回：测试副本中允许编辑文本线索的 `Text` 列，按原偏移原容量进行 GBK 等长/缩短写回；保存前备份，保存后复读验证。原始目录仍只读，暂不支持扩容、搬移文本区或完整命令树写回。
