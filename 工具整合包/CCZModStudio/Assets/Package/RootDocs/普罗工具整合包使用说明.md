# 普罗工具整合包隐藏用法速查

## 通用表格快捷键

- 表格复制：选中表格区域后按 `Ctrl+C`；复制不带表头的单元格文本。
- 表格粘贴：选中起始单元格后按 `Ctrl+V`；按剪贴板行列覆盖可写单元格。
- 批量填列：选中目标区域后按 `Ctrl+D`；用当前单元格值填充同列选区。
- 批量修改：在表格中按 `Ctrl+H`；打开当前表格的批量修改窗口。
- 清空单元格：选中可写单元格后按 `Delete`；只清空可编辑区域。
- 直接编辑：选中可写单元格后按 `F2`；进入当前单元格编辑态。
- 撤销重做：表格内按 `Ctrl+Z`、`Ctrl+Y` 或 `Ctrl+Shift+Z`；撤销或重做表格编辑历史。
- 搜索确认：角色、兵种、宝物、商店、图片资源等搜索框输入后按 `Enter`；直接执行当前筛选。
- 数字进制：顶部 `10/16` 只影响当前页数字显示；不改变写回数值。

## 兵种与策略

- 可装备类别：单击详细兵种表的 `可装备类别` 列；右侧显示可装备类别复选框。
- 可装备类别同步：在右侧勾选类别后切换到其他单元格或保存兵种；同步回详细兵种主表摘要。
- 可装备类别提示：悬停右侧类别复选框；查看预览说明和底层原始列。
- 兵种策略学习：单击兵种策略表的 `名称` 列；右侧显示该策略对 80 个兵种的学习等级表。
- 学习等级规则：等级 `0` 表示该兵种不能学习该策略；`1..255` 表示学会等级。
- 学习摘要同步：在右侧修改学习等级后切换到其他单元格或保存兵种策略；同步回主表可学摘要。
- 兵种形象替换：一键替换兵种形象时只写入弹窗中勾选的阵营槽；未勾选阵营保持不变。
- 兵种预览联动：选择攻击范围、穿透、策略范围、策略图标或动画字段；右侧预览切换到对应资源。

## 宝物与图片资源

- 宝物图标缩放：在大图或小图预览上滚轮；缩放当前预览。
- 宝物图标复位：双击大图或小图预览；恢复该预览缩放。
- 宝物图标批量导入：批量写入目标图标资源；不会修改宝物表里的“图标”字段。
- 图片批量匹配：E5 批量导入按文件名数字或选中行匹配目标图号；文件名建议包含目标编号，例如 `12.png` 或 `#12.png`。
- DLL 图标恢复：DLL 图标不能用 E5 还原按钮恢复；需用保存前备份恢复整文件，或重新导入图片。
- R/S 默认规则：人物 `S=0` 按职业和预览阵营取默认兵种图；`S=1..32` 取三转特殊三张图；`S>=33` 取一转特殊单张图。
- R/S 缺失检查：勾选“仅缺失资源”；只显示 R 或 S 资源无法定位的角色。
- S 预览阵营：切换预览阵营；只改变 `S=0` 默认兵种图的预览，不直接改人物表。

## 人物形象导入导出格式

- 头像一键导入：选择单张 `BMP/JPG/PNG`；写入当前行头像号，导入时居中裁切并缩放为 `Face.e5` 的 `120x120 PNG`，不修改人物表“头像”字段，不写 `Tou.dll` 真彩头像。
- 头像批量导入：可多选图片或选择导出根目录；文件名 `face_编号.bmp/png/jpg` 按头像编号匹配，无法匹配时按选中行与文件数量做顺序匹配。
- 头像批量导入旧目录：选择文件夹时也识别 `face_编号...` 子目录；目录内优先取 `face.bmp`，头像 `0` 号目录还识别 `face_01.bmp`。
- 头像重复限制：一次导入中多个选中行不能指向同一头像编号；重复头像编号会阻止写入。
- 头像导出：单行导出在所选目录直接生成 `face_编号.bmp`；多行导出在同一目录平铺生成多个 `face_编号.bmp`。
- 头像 0 号边界：头像号 `0` 是 `Face.e5 #1-#8` 遗留多表情槽；当前导入/导出只处理 `#1`，不自动覆盖或导出 `#2-#8`。

- R 一键导入：选择包含 `front.bmp` 与/或 `back.bmp` 的目录；也识别 `x.bmp/1.bmp` 作为正面、`y.bmp/2.bmp` 作为反面，文件名不区分大小写。
- R 图片尺寸：正面和反面 BMP 均必须为 `48x1280`；内部按 `20` 帧解释，每帧 `48x64`。
- R 图号映射：`R=n` 写入 `Pmapobj.e5 #(2n+1)` 正面和 `#(2n+2)` 反面；一键导入允许只导入存在的一侧。
- R 批量导入：选择素材根目录；子目录名必须为 `编号`、`R编号` 或 `R_编号`，例如 `12`、`R12`、`R_12`。
- R 批量导入完整性：每个 R 子目录必须同时包含正面和反面文件；缺任一侧、编号重复、目标图号越界会阻止或跳过写入。
- R 导出：单行导出在所选目录生成 `front.bmp`、`back.bmp`；多行导出生成 `R编号` 子目录，每个子目录内为 `front.bmp`、`back.bmp`，可直接作为批量导入素材。

- S 一键导入：选择包含 `mov.bmp`、`atk.bmp`、`spc.bmp` 的目录；一键导入允许只导入存在的动作文件。`S=1..32` 可在弹窗中选择第一转/第二转/第三转，默认只勾选第一转；单转 S 只显示第一转。
- S 分转素材：三转 S 可使用 `turn1/mov.bmp`、`turn2/mov.bmp`、`turn3/mov.bmp` 子目录；导入第 N 转时优先读取 `turnN`，缺少对应文件时回退读取 S 目录根部的同名文件。
- S 图片尺寸：`mov.bmp` 必须为 `48x528`，`atk.bmp` 必须为 `64x768`，`spc.bmp` 必须为 `48x240`。
- S 图号映射：`S=0` 按当前人物职业和预览阵营取默认兵种图；`S=1..32` 为三转特殊 S，第一/二/三转分别对应三个 Unit 图号；`S>=33` 取一转特殊单张图。
- S 批量导入：选择素材根目录；子目录名必须为 `编号`、`S编号` 或 `S_编号`，例如 `12`、`S12`、`S_12`；批量前可选择要导入的转。
- S 批量导入完整性：每个实际写入的转必须同时包含或回退匹配到 `mov.bmp`、`atk.bmp`、`spc.bmp`；缺文件、编号重复或解析到重复 Unit 图号会阻止写入。
- S 导出：单行导出默认在所选目录生成 `mov.bmp`、`atk.bmp`、`spc.bmp`；导出多个转或第二/第三转时生成 `turn1` / `turn2` / `turn3` 子目录。多行导出生成 `S编号` 子目录，三转导出时目录形如 `S1/turn2/mov.bmp`，可直接作为批量导入素材。
- S 多图导出边界：一个 S 映射到多个 Unit 图号时，可在导出前选择要导出的转；单转 S 只会导出第一转，并对忽略的第二/第三转给出提示。
- 导出覆盖规则：单行导出若目标目录已有同名 BMP，会询问是否覆盖；选择不覆盖时跳过已有文件。

## 地图编辑画布

- 素材库自动定位：进入地图编辑、读取地图或使用素材功能时；自动尝试项目、工作区、旧工具和随包资源里的常见素材库目录。
- 素材层规则：地形素材按 `HexTag` 生成地形层；建筑作为 48x48 覆盖层；景物作为可移动覆盖物。
- 矩形选区：按住 `Shift` 左键拖拽地图格；选择一片矩形区域。
- 提取素材：右键当前格或先框选区域后右键；把底图选区提取到素材库。
- 地图绘制撤销：一次连续拖拽绘制会合并为一条历史；撤销时整笔回退。
- 景物选中：点击景物覆盖物；显示移动、缩放、旋转控制点。
- 景物移动：选中景物后按方向键；每次移动 1 像素，`Shift+方向键` 每次移动 10 像素。
- 景物缩放：选中景物后按 `+` 或 `-`；小步缩放，`Shift++` 或 `Shift+-` 大步缩放。
- 景物旋转：选中景物后按 `[` 或 `]`；每次旋转 5 度，`Shift+[` 或 `Shift+]` 每次旋转 15 度。
- 景物删除：选中景物后按 `Delete` 或右键该景物；删除当前覆盖物。
- 美化回退：生成美化预览后点击回退美化；恢复未美化的素材绘制预览。
- 发布边界：一键发布会写入地图图片；地形层只在能匹配 Hexzmap 地形块时同步发布。

## 剧本树高级编辑

- 命令双击：双击剧本树命令；打开旧版 Dialog 或当前可用的参数编辑。
- 参数双击：双击右侧参数表；进入选中参数的修改流程。
- 参数回车：在参数值输入框按 `Enter`；执行当前参数修改。
- 右键选中：在剧本树节点上右键；先选中该节点，再打开对应上下文菜单。
- 快捷修改：选中命令后按 `Ctrl+E`；修改当前命令。
- 上方添加：按 `Ctrl+Shift+I`；按当前节点类型在上方添加 Scene、Section 或指令。
- 下方添加：按 `Ctrl+I`；按当前节点类型在下方添加 Scene、Section 或指令。
- 子事件上方添加：按 `Ctrl+Shift+O`；在支持的位置上方插入子事件。
- 子事件下方添加：按 `Ctrl+O`；在支持的位置下方插入子事件。
- 步进复制：按 `Ctrl+D`；复制当前命令并按指定槽位递增参数。
- 步进复制限制：不支持 `46/47/5/0/1/2` 号指令，不支持带子事件块的指令，不支持变量数组槽。
- 删除命令：选中节点后按 `Delete`；删除当前可删命令、Scene 或 Section。
- 移动命令：按 `Ctrl+Up` 或 `Ctrl+Down`；移动当前命令位置。
- 剪切复制粘贴：按 `Ctrl+X/C/V`；支持命令、Scene、Section 和多选命令。
- 粘贴预览：粘贴前查看预览；确认粘贴后仍需完整保存剧本才写入文件。
- 全部展开：按 `Ctrl+Q`；展开当前剧本树节点。
- 撤销前进：按 `Ctrl+Z` 或 `Ctrl+Y`；撤销或重做剧本结构编辑。
- 跳转目标：对 `0x76` 等可跳转命令使用右键“跳转到”；定位关联命令或对象。
- 文本导入：右键目标命令选择“文本导入...”；解析成功后把新命令插入到目标下方，仍需完整保存剧本才写入文件。
- 搜索定位：双击搜索结果；跳到对应剧本树节点。

## 场景编辑画布

- 动作帧放置：从动作帧列表拖到画布；在背景格子上放置角色动作帧。
- 画布角色移动：拖动画布上的角色；更新场景草稿位置。
- `0x30` 同步：拖动绑定 `0x30 武将出现` 的角色；同步该命令的 X/Y 坐标。
- 非 `0x30` 边界：拖动其他视觉命令角色；只保存到项目侧场景草稿。
- 角色定位：双击画布角色；定位到关联 R 剧本命令。
- 右键编辑：右键画布角色；进入当前角色编辑态。
- 场景缩放：在场景画布上滚轮；调整缩放，`Ctrl+滚轮` 大步缩放。
- 预览锁定：锁定预览后；切换命令时保持当前预览范围。
- 播放间隔：修改停顿毫秒数；影响场景预览步进速度。

## 战场编辑画布

- 拖拽出场：从角色列表拖到地图格；生成一条手工摆放记录。
- 已摆放编辑：右键地图上的已摆放单位；进入可编辑状态。
- 出场预览同步：拖动已摆放单位；同步 `46/47/4B` 出场设置预览。
- 状态弹窗：双击地图上的友军或敌军；打开战场单位状态编辑。
- 候选定位：双击候选表行；跳到对应 S 剧本命令。
- 命令定位：双击命令表行；跳到对应 S 剧本命令。
- 地图缩放：在战场地图上滚轮；调整缩放，`Ctrl+滚轮` 大步缩放。
- 控制面板联动：修改阵营、隐藏、等级修正、兵种级、AI 方针或方向；自动应用到当前选中摆放单位。
- 出场写回边界：写回出场到 S 剧本只写已确认的 `46/47/4B` 槽位；等级、装备和未知状态槽保持原值。
- 单位状态边界：状态弹窗里 `52` 兵种和 `38` 五维属于脚本运行指令；不是永久人物表字段。

## 弹窗快捷键

- 像素编辑保存：按 `Ctrl+S`；确认保存写回当前图片。
- 像素编辑导入：按 `Ctrl+I`；导入 PNG 到当前画布。
- 像素编辑导出：按 `Ctrl+E`；导出当前画布 PNG。
- 像素编辑撤销：按 `Ctrl+Z`；撤销一步。
- 像素编辑重做：按 `Ctrl+Y` 或 `Ctrl+Shift+Z`；重做一步。
- 像素编辑网格：按 `Ctrl+H` 或 `G`；显示或隐藏网格。
- 像素编辑工具：按 `P/E/I/F/L/R`；切换铅笔、橡皮、取色、填充、直线、矩形。
- 像素编辑缩放：按 `+` 或 `-`；缩放画布，按 `0` 重置缩放。
- 剧本变量刷新：在剧本变量弹窗按 `F5`；刷新当前页或项目扫描。
- 剧本变量定位：在剧本变量弹窗按 `Enter` 或双击记录；定位出现位置。
- 兵种形象替换取消：在兵种形象替换弹窗按 `Esc`；取消本次替换。

## 写回边界

- 表格类保存：保存前自动备份目标文件；保存后重新读取校验。
- 完整保存剧本：会重建 Scene 偏移、Section/子块长度和 `0x76` 跳转。
- 单独保存文本：只写已匹配文本线索；未知命令结构保持原样。
- 文本容量：修改剧本文本时注意字节容量提示；超出容量会阻止写回或要求改短。
- 战场标题胜败：只写已匹配的标题和胜败条件文本；未知命令结构保持原样。
- 战场出场核对：候选标记只保存核对状态；写回 S 剧本需另行执行出场写回。
- 图片资源写入：E5、DLL、RAW 等资源写入前会预览并备份；不同格式不能混用还原方式。
- 草稿与正式文件：地图草稿、场景草稿不会自动改游戏文件；发布或完整保存后才写入项目文件。
