## 详细记录

# CCZModStudio 本地知识库

本知识库用于记录曹操传 模组 制作工具整合过程中已经确认的格式、工具来源、实现约束和待补充知识。

## 使用规则

1. 后续开发前先查看本知识库，再查看 `工具整合包\plan.md`。
2. 用户纠正格式、工具来源、传统制作习惯时，优先写入本知识库，再调整代码和 `plan.md`。
3. 未确认的格式不能凭待证假设写回；必须标为“待查”，并记录需要用户提供的来源工具或样例文件。
4. 本知识库是项目长期上下文：每次修复格式误判、读写边界、核心工具行为时都要更新。
