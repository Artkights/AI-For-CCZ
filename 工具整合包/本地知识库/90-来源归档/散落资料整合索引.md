# 散落资料整合索引

## 结论速览

- 本页登记当前工作区内尚未完全纳入主知识库的 MOD 知识来源、处理优先级和去向。
- 原始资料默认保留原位；本页只记录来源、版本边界、证据等级和归入主题。
- 第一批整合以 6.5 实作为主：`特效整理/6.5`、根目录剧本模板、CCZModStudio/MCP README、旧工具关键配置。

## 适用版本

- 默认适用：曹操传加强版 6.5（未加密）当前项目基底。
- 6.6/6.6x、6.4、原版和其它 MOD 样本只作为版本差异、样本对照或待验证线索。

## 已确认事实

- 主知识库位于 `工具整合包/本地知识库`。
- 当前知识库已有约 70 个 Markdown 和 3 个 docx；分层目录完整，但仍存在散落资料。
- 高噪声目录和文件很多，必须先排除构建产物、临时文件、备份、日志和二进制文件。

本批自动生成的工作草稿位于 `90-来源归档/制作区帖子备份-抽取草稿/`：

- `sources.csv`：60 个已展开文件的路径、类型、大小、mtime、SHA256、主题、归口和处理状态；不包含 `star175.rar`。
- `topic-map.md`：按 EXE 逆向、star175、godtype、特效、剧本变量和资源美工分流。
- `extract-summary.md`：每个来源的短摘要、候选地址/变量/资源和复用边界。
- `unresolved.md`：legacy `.doc` 待抽取、空 txt、疑似分页/重复主题和验证队列。

处理原则：本批资料不能直接升级为 6.5 写回规则；地址类必须先做当前 `Ekd5.exe` 静态复读或 x32dbg 动态命中，剧本类必须对照 `.eex` 样本和旧剧本编辑器源码，资源类必须做 E5/DLL 读取验证，特效类必须补当前 6.5 触发链或烟测。
