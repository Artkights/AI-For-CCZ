# 迁移索引

## 结论速览

- 本页记录旧平铺知识库和嵌套 `knowledge-base` 到新分层目录的迁移关系。
- 旧文件不再作为主入口；查阅时从根 `README.md` 进入。
- 原始 docx 保存在 `90-来源归档/原始文档`，只作为来源，不直接替代结构化专题。

## 适用版本

- 默认版本：曹操传加强版 6.5（未加密）。
- 旧入口和外部文档中涉及的其它版本只作为来源归档。

## 已确认事实

- 旧根目录平铺 Markdown 已迁移到分层目录。
- 嵌套 `knowledge-base` 已提升到根目录相同分类下。
- 旧入口 README 已归档为 `旧入口-README-20260606.md` 和 `旧入口-knowledge-base-README.md`。

## 实现/使用方法

- 查旧文件去向时使用下方映射表。
- 新增内容必须写入分层目录主题文档，不再新增根目录编号 Markdown。
- 若外部脚本仍引用旧路径，应改为新路径或通过根 README 导航。

## 风险与边界

- 迁移保留了历史细节，但统一模板里的“结论速览”和“风险与边界”优先于旧正文。
- 原始 docx、论坛帖和旧入口只作证据来源，不作为主流程入口。
