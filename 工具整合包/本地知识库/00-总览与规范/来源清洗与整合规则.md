# 来源清洗与整合规则

## 结论速览

- 本页定义散落 MOD 知识进入本地知识库前的收录、排除、去重、编码和证据规则。
- 当前整合目标是“去除脏数据后归纳进现有知识体系”，不是搬迁或复制所有原始文件。
- 默认优先级为 6.5 实作优先：能服务当前未加密基底写回、剧本、资源、特效和工具链的资料先处理。
- 6.6/6.6x、6.4、原版和其它 MOD 样本必须保留版本护栏，不能混入 6.5 默认写入路径。

## 适用版本

- 默认适用：曹操传加强版 6.5（未加密）当前项目基底。
- 其它版本资料只作为版本差异、外部佐证、样本对照或待验证线索。

## 已确认事实

- 主知识库固定为 `工具整合包/本地知识库`。
- 原始散落资料默认保留原位，知识库只写规范化结论、证据摘要、来源索引和待验证项。
- 当前主要来源包括根目录文档、`特效整理`、`工具整合包` 非知识库资料、`老版游戏制作工具`、`基底`、`CCZModStudio_Notes`。
- `_BuildCheck`、`_codex_tmp`、`bin`、`obj`、`.vs`、构建日志、备份和临时文件是高噪声来源，默认排除。

## 实现/使用方法

### 收录分流

| 分类 | 定义 | 处理方式 |
|------|------|----------|
| 收录 | 6.5 实作相关，来源清楚，可归入现有主题 | 抽取结论写入主题文档，来源写入证据表 |
| 仅索引 | 有检索价值，但不适合写正文 | 写入 `90-来源归档/散落资料整合索引.md` |
| 归档待证 | 有线索价值但缺少本地验证或版本不匹配 | 只写边界和待验证项，不作为实现规则 |
| 排除脏数据 | 构建产物、临时文件、备份、失败日志、二进制、重复发布包 | 不进入主题正文；必要时只在排除记录中说明 |

### 来源优先级

1. 第一批：`特效整理/6.5`、根目录提示词与剧本模板、`CCZModStudio` / MCP README、旧工具关键配置。
2. 第二批：`老版游戏制作工具` 中的剧本编辑器、形象指定器、`CczRSX 6.5`、地形编辑器、扳手、普罗注入资料。
3. 第三批：`基底` 中的 6.5 样本、安西军传和三英战龙帝制作记录、可复读的地图与 R/S 案例。
4. 第四批：`CCZModStudio_Notes` 中的烟测、`EffectManifests`、`MapWorkbenchDrafts`、调试草稿；只抽取稳定结论，备份和失败样本不进正文。

### 脏数据排除规则

- 默认排除目录：`.git`、`.vs`、`_BuildCheck`、`_codex_tmp`、`bin`、`obj`、`.dotnet_home`、`.appdata`、`.localappdata`、`.userprofile`、发布输出目录。
- 默认排除文件：`*.dll`、`*.exe`、`*.pdb`、`*.log`、`*.bak*`、`*backup*`、`*tmp*`、`*corrupt*`、`*HEAD*`、`build_errors*.txt`、`build_after_repair*.txt`。
- 失败日志、备份和临时文件可以用于定位问题，但不得作为“已验证结论”的证据来源。
- 如必须引用失败样本，必须放在“风险与边界”或“待验证项”，并标明它不能作为写回依据。

### 去重和主题映射

- 同一主题只保留一个主文档；新增信息应合并到已有专题，不新建平行专题。
- 旧入口、重复模板、根目录副本和同名变体只保留来源索引。
- 主题映射优先遵循根 README 的分层目录：引擎进 `01/03/04`，数据结构进 `02`，工具链进 `06`，资源表进 `07`，剧本战场进 `08`，版本差异进 `09`，项目案例进 `10`。

### 编码和格式

- Markdown 统一使用带 BOM 的 UTF-8。
- 文档标题、导航、说明、表格字段使用中文；技术标识、地址、函数名、文件名和命令参数保持原文。
- 新增条目必须使用 `知识条目模板.md` 的七段结构。
- 原始路径写相对路径；除非说明“旧口径”，不得保留 `D:\Downloads` 等旧个人目录。

## 风险与边界

- 来源清洗不等于验证。能被索引只代表可查，不能自动成为可写回规则。
- 其它 MOD 基底中的经验可能受版本、补丁、资源槽、剧本结构影响；必须通过当前 6.5 基底复读或烟测后才能升级。
- `CCZModStudio_Notes` 中大量 `.bak.json` 和失败烟测记录只能辅助排查，不能直接进入正文。
- 6.6/6.6x 特效和资源口径只进入版本差异或未来适配，不覆盖 6.5 默认路径。

## 证据来源

- 统计来源：当前工作区只读盘点，见 `../90-来源归档/散落资料整合索引.md`。
- 关联规范：`知识库规范.md`、`知识条目模板.md`、`开发执行规则.md`。
- 证据等级：已验证；本页是基于当前工作区盘点形成的内部清洗规则。
