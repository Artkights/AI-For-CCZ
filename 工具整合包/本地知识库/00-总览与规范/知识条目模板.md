# 知识条目模板

## 结论速览

- 本页提供新增或重建知识条目时的固定模板。
- 新条目必须先说明能直接用于 6.5 实作的结论，再说明版本边界和证据等级。
- 缺少本地样本、旧工具源码、复读、烟测或实机验证时，不能写成可写回规则。

## 适用版本

- 默认适用：曹操传加强版 6.5（未加密）当前项目基底。
- 6.6/6.6x、6.4、原版或其它 MOD 资料只作为版本差异、外部佐证或待验证线索。

## 已确认事实

- 每个主题只维护一个主文档；重复来源进入证据表或 `90-来源归档`。
- 原始散落资料默认保留原位，知识库只沉淀规范化结论、索引和证据摘要。
- 文档必须使用当前知识库相对链接，不保留旧盘符、旧下载目录或无法解析的跳转。

## 实现/使用方法

复制下面模板创建新条目：

```markdown
# 标题

## 结论速览

- 用 2-5 条写清本页最重要、最可执行的结论。
- 明确哪些结论可以直接服务 6.5 实作，哪些只是线索。

## 适用版本

- 默认适用：曹操传加强版 6.5（未加密）当前项目基底。
- 其它版本：写清版本号、来源和是否能迁移。

## 已确认事实

- 只写已经由本地样本、旧工具源码、复读、烟测或实机确认的事实。
- 若来自散落资料，写清原始路径和抽取范围。

## 实现/使用方法

- 写具体操作、查询入口、字段口径、流程或命令。
- 写入类规则必须包含目标版本、目标路径、备份、复读和失败处理。

## 风险与边界

- 写清不能直接套用的版本、未确认字段、冲突补丁、容量限制和脏数据来源。
- 高风险或缺证据内容只放入待验证项，不进入默认流程。

## 证据来源

- 本地来源：`相对/路径/文件名`，证据等级：已验证 / 外部佐证 / 待证 / 废弃或旧口径。
- 关联专题：`../xx-分类/专题.md`。

## 待验证项

- 列出进入实现前必须补齐的样本、复读、烟测、实机或源码验证。
```

## 风险与边界

- 模板中的示例文本必须替换成实际结论，不能保留占位符进入正式专题。
- 旧工具、论坛资料、其它 MOD 样本和 6.6/6.6x 补丁不能直接写成 6.5 默认规则。
- 烟测失败、构建日志、备份文件和临时目录只能作为问题排查记录，不能作为正式证据。

## 证据来源

- 关联规范：`知识库规范.md`、`来源清洗与整合规则.md`。
- 迁移与来源索引：`../90-来源归档/迁移索引.md`、`../90-来源归档/散落资料整合索引.md`。
- 证据等级：已验证；本页是知识库内部模板规范。

## 待验证项

- 后续如新增自动化检查脚本，需要把模板章节完整性纳入校验。
