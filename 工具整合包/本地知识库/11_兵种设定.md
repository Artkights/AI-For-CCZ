## 当前实现

- `兵种设定` 页面包含三个创作向子页：
  1. `详细兵种`：合并名称、说明、成长、穿透。
  2. `兵种系/地形`：合并兵种系名称、地形发挥、移动消耗。
  3. `兵种特效`：合并特效说明、分配，并只读显示人物名/兵种名。
- `详细兵种` 右侧已新增范围预览：选中 `攻击范围` 单元格时按字段值 + 1 读取 `E5\Hitarea.e5` 图号；选中 `穿透` 单元格时按字段值 + 1 读取 `E5\Effarea.e5` 图号。预览只读，不改 E5 资源。
- 保存策略：保存前自动备份，保存后复读校验。
- `6.5-7 兵种特效` 名称表在 `HexTable.xml` 中 `Bytes=0`，当前不开放改名；只读读取名称，避免破坏未拆清结构。

### 联网资料归纳

- 联网检索目标：曹操传 MOD 的 `攻击距离/攻击范围`、`Hitarea.e5`、`Effarea.e5`、兵种穿透资料。
- 可迁移结论：
  - 攻击距离是兵种成长表中的单字节字段，和本地 `6.5-4-2 兵种成长` 的 `攻击范围` 列一致。
  - 6.5 范围图片资源口径：攻击范围看 `Hitarea.e5`，穿透范围看 `Effarea.e5`；字段值是编号，图片资源提供可视化形状。
  - 旧资料对 0..10 攻击距离的常见解释可作为 UI 提示，但不能替代实机验证：0 十字相邻；1 周围一圈；2 一转弓兵/二转弓骑候选；3 二转弓兵/三转弓骑候选；4 三转弓兵候选；5 原版常规兵种少见；6 没羽箭；7 一转炮车；8 二三转炮车；9 一转弓骑兵；10 全屏。10 以后在 6.5 的 `Hitarea.e5` 仍有扩展图，语义需实机验证。
  - 旧资料对穿透的常见解释可作为 UI 提示：0 正常攻击；1 十字穿透；2 九宫穿透；3 大没羽箭；4 蛇矛；5 长蛇矛/穿六；6 大大没羽箭。6 以后在当前 `Effarea.e5` 仍有扩展图，语义需实机验证。
- 资料边界：
  - 外部资料只作为字段含义和资源口径的参考。真正写入仍以 `HexTable.xml`、本地 E5 索引读取、保存前备份、保存后复读和实机战场测试为准。
  - 当前工具只预览范围形状，不修改 `Hitarea.e5` / `Effarea.e5`。
