## 详细记录

#### 概述

曹操传中存在一种"攻击命中后使目标随机获得异常状态"的能力，其个人特效号为 `0x0B` (11)，装备特效号为 `0x5E` (94)。该能力的检测点位于 `0x405A59` 附近，桩函数内联在 `0x472C41` 中。

**四种异常状态**：中毒、麻痹、禁咒、混乱。

**关键特征**：与属性下降攻击（0x472D48）类似，该能力同样使用内联桩函数架构，位于后伤害效果处理流程中。

---

#### 完整调用链

```
后伤害效果调度器 0x405966 (attacker, target)
    │
    ├── 0x40399B: 伤害效果执行
    ├── 0x472C41: ★ 随机状态攻击（本文档所述）← 判定点 0x405A59
    ├── 0x472D48: 属性下降效果
    ├── 0x4039F4: 伤害动画/音效
    ├── 0x436392: 前置条件检查
    └── ...
```

#### 入口函数 0x472C41 详细分析

##### 函数概览

| 属性 | 值 |
|------|-----|
| 地址 | `0x472C41` |
| 功能 | 随机状态攻击特技入口 |
| 调用者 | `0x405966` (后伤害效果调度器) |
| 个人特效号 | `0x0B` (11) |
| 装备特效号 | `0x5E` (94) |

##### 预期汇编结构（类比属性下降）

```asm
; 0x472C41: 后伤害随机状态攻击入口
; 参数: [ebp+0x08] = attacker_ptr (攻击者)
;       [ebp+0x0C] = target_ptr  (被攻击者)
0x472C41: push ebp
0x472C42: mov ebp, esp
          ...
          ; 前置条件检查
          ...
          
; ===== 内联桩函数: 检测 0x5E/0x0B 特技 =====
          mov ecx, [ebp-0x08]     ; ecx = attacker
          push 0x01               ; 效果值标志 = 1 (无效果值)
          push 0x01               ; 叠加标志 = 1 (不可叠加)
          push 0x5E               ; ★ 装备特效号 = 0x5E (94)
          push 0x0B               ; ★ 个人特效号 = 0x0B (11)
          call 0x600xxx           ; → 调度函数
          test eax, eax
          jz 0x472Cxx             ; 无特技 → 退出
          
; ===== 随机状态应用 =====
          push 0x04               ; 参数 = 0x04 (4种异常状态)
          call 0x47FB13           ; ★ 随机选择状态索引 (0-3)
          ...
          ; 应用异常状态到目标
          ...
          
0x472Cxx: mov esp, ebp
0x472Cxx: pop ebp
0x472Cxx: ret 0x08
```

#### 四种异常状态详解

##### 状态位编码

异常状态存储在战术单位的 `0x1E` 偏移处（状态标志字节）：

```
unit[0x1E] = 状态标志字节
```

###### 位结构

| 位 | 含义 |
|----|------|
| bit 0-1 | 状态类型 |
| bit 2-3 | 状态等级/强度 |
| bit 4-5 | 麻痹/禁咒状态 |
| bit 6-7 | 中毒状态 |

###### 四种状态对应

| 状态 | 位掩码 | 效果 |
|------|--------|------|
| 中毒 | `0xC0` (bit 6-7) | 每回合损失HP |
| 麻痹 | `0x30` (bit 4-5) | 无法行动 |
| 禁咒 | `0x0C` (bit 2-3) | 无法使用策略 |
| 混乱 | `0x03` (bit 0-1) | 行动不受控制 |

#### 与属性下降攻击的对比

| 特性 | 属性下降攻击 (0x472D48) | 随机状态攻击 (0x472C41) |
|------|------------------------|------------------------|
| 个人特效号 | `0x0C` (12) | `0x0B` (11) |
| 装备特效号 | `0x5E` (94) | `0x5E` (94) |
| 影响目标 | 属性增益档位 | 异常状态标志 |
| 目标偏移 | `0x21-0x26` (属性数组) | `0x1E` (状态字节) |
| 随机选项数 | 6种属性 | 4种状态 |
| 应用方式 | 属性值调整 | 位标志设置 |
| 判定点 | `0x472D63` | `0x405A59` |

---

#### 特技参数表

| 参数 | 值 | 含义 |
|------|-----|------|
| 个人特效号 | `0x0B` (11) | 角色天生状态攻击能力 |
| 装备特效号 | `0x5E` (94) | 装备提供的状态攻击 |
| 状态数量 | `0x04` | 4种异常状态 |
| 桩函数位置 | `0x472C41` | 内联在后伤害效果中 |

#### 设计注意事项

若要在Mod中修改此机制：

1. **修改状态类型**：调整随机选择的状态范围
2. **添加新状态**：扩展 `0x1E` 字节的位定义
3. **修改命中概率**：调整随机判定逻辑
4. **状态持续回合**：修改状态持续时间相关代码
