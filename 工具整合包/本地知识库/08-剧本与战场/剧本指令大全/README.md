# 剧本指令大全

## 结论速览

- 本目录是 R/S `.eex` 剧本指令的多页速查与逐参数核对手册，默认服务于曹操传加强版 6.5（未加密）。
- 手册覆盖 `0x00-0x7B` 全部旧剧本编辑器命令，并单列 `0x72 信息传送` 子功能。
- 每条命令都给出一句话用途、常用场景、参数槽、旧源码布局、子事件能力、写回边界和证据来源。

## 适用版本

- 默认适用：曹操传加强版 6.5（未加密）当前项目基底。
- 6.6/6.6x 内容只作为版本差异和只读附注，不进入 6.5 默认写回规则。

## 已确认事实

- 命令名来自 v0.23 `CczString.ini`，范围为 `0x00-0x7B`。
- 参数槽优先来自 `ScenarioCommandParameterTemplateService.cs`，模板缺失时只列旧源码 `code_instruct`。
- 子事件能力来自 v0.23 `cczEditor2View.h` 的 `code_test`。
- `46` 为 20 组友军部署记录，每组 11 槽；`47` 为 80 组敌军部署记录，每组 12 槽。
- `76` 跳转保存时按目标命令偏移回填，不能手填普通编号。

## 实现/使用方法

1. 按命令号进入对应分页。
2. 先读“一句话用途”和“常用场景”，再核对参数槽与旧源码布局。
3. 写回前检查“写回边界”；凡标高风险或待验证的命令，只能先做样本复读和实机验证。
4. `0x72` 相关功能先看 `08-72信息传送子功能.md`，不要把 `72` 文本当普通对白替换。

## 风险与边界

- 本目录是知识库手册，不是写回工具。
- 不把外部教程或 6.6 配置直接当作 6.5 写入规则。
- 不把旧源码类型码等同于完整业务语义；未知参数必须保留待验证。
