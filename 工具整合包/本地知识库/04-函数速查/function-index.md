#### 调用层级索引

##### 物理攻击 → 引导攻击

```
0x43DF0A (行动类型分发器)
    └── 0x43DADA (行动分发入口)
        └── 0x4212F1 (包装函数) / 0x43AAF7 (主编排器)
            ├── 0x417C67 (调用者1 - 目标列表构建)
            └── 0x436499 (获取目标列表)
```

##### 行动结束 → 二次行动（仅特技持有者触发）

```
行动结束
    └── 0x412D52 (二次行动判定与执行)
        ├── 0x4061F9 (获取单位指针)
        ├── 0x421175 (桩函数判定)
        └── 0x406690 (恢复行动能力)
```

##### 回合开始 → 重置

```
新回合
    └── 0x44AF8F (重置循环)
        ├── 0x4061F9 (获取单位指针)
        └── 0x406690 (恢复行动)
```
