简要说明：
修改器可能存在一些未知BUG，所以使用前请先备份相关文件

使用须知:
指定目录，读入hexzmap.e5
请不要在RPGViewer、游戏启用情况下使用导入功能
地图的最大横、纵坐标必须在1-85之间
左键是点操作，右键是点的还原操作
选择地形后，按住鼠标左键在图上拖曳，可快速填充地形数据
按住鼠标右键在图上拖曳，可快速还原地形数据

1.04 
支持地图的导入、导出
导入原理：
导入一幅位图文件.bmp，获取位图颜色信息，替换或追加Spalet.e5中数据，未根据.bmp的长宽改变hexzmap.e5中相应的长宽比。
生成Hm??.e5，改动Spalet.e5，导入后，请根据导入的Hm??.e5长宽自行调整Hexzmap.e5中的长宽
导出原理：
根据Hexzmap对于地图的长宽定义，获取Spalet.e5中对应的颜色表，将hm??.e5导出成.bmp文件
